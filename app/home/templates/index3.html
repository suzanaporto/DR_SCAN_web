About

REGULOMIX is a user-friendly web service prepared to suggest valuable potential candidates of cis or trans-regulatory elements from an SNP-panel of interest in a regulomics fashion. It is a powerful tool for investigating the role of non-coding DNA in regulatory mechanisms related to complex traits. The integrated pipeline of REGULOMIX consists of 5 steps.

Step 1- Enter the SNPs. You will provide SNPs related a complex trait of your interest into the software. To enter the SNPs you can put dbSNP id into the first text field and click on the submit button beside or make an excel file (.xlsx) with only one column with header named SNPs and write in each row one SNP ID, upload the file and then click on the submit button. The software will process each SNP and show the user a table below the inserted SNP form containing each SNP and its information of location (GRCh37), chromosome, wild-type allele and variation allele. Sometimes there can be more than one variation allele so they are separated with a vertical bar. Each row is provided with one edit button and one delete button. Delete button will remove the row and edit button can be used if some info is missing from the table, the user can click once to edit and click again to save the modifications.
Every step is bound by step 1. If you upload a new excel file, all the other steps are going to be deleted.
Step 2 – Scan for Cis-Regulatory Elements. Insert the id of the tissues they want to analyze. You can write more than one separated with a vertical bar. In the end, a table will be shown with the results. You can delete SNPs in this step, choosing the labels it wants to keep, or by providing a list of SNP IDs into the text field separated by comma. The generated analysis from this step can be downloaded in .csv format before next step.
Step 3 – Choose Transcription Factor Matrix. Choose one of the provided matrixes or uploads one to use. This step also shows the sequences made with all remaining SNPs in FASTA format.
Step 4 – Differential Transcription Factors. A table is showed to the user with all the differential transcription factors identified by the software. The results from this step can be downloaded in .csv format before next step.
Step 5 – Detection of Enhancer-Promoter Interactions. The user will provide tissues IDs separated by a vertical bar and it will show a table with all the enhancer-promoter interactions. All resulted tables from each step can be downloaded in .csv format with a button on the top of the table.
REGULOMIX also contains a database to keep the current processing of each one of the steps, so it can store results if the processing is longer and keep it available for 7 days, unless the user starts a new process.

The Noisy Neighbors Algorithm
If the SNP is too close to other SNP (< 31 bp), they will be put together to generate sequences with the combination of their alleles. This is based on the fact that more than one SNP can be inherited together and the recent work that showed that haplotypes can explain better genetic inheritance in complex diseases like multiple sclerosis (KHANKANIAN et al., 2015). In this context, we have created an original algorithm to construct these combinatory sequences and it was implemented in REGULOMIX. Therefore, some outputs of our tool may have results with a sequence that has more than one SNP. It is important to highlight that the results obtained through the application of Noisy Neighbors Algorithm are based uniquely on the distances between two regulatory SNPs, being necessary to verify the occurrence of linkage disequilibrium in population genomic data.

Reference
Khankhanian P, Gourraud P, Lizee A, et al. Haplotype-based approach to known MS-associated regions increases the amount of explained risk. Journal of Medical Genetics 2015;52:587-594.
